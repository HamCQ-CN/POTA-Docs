# 颁奖活动

自2018年以来，空中公园每年都会举办一个特别的颁奖活动，参与者根据活动情况，有机会获得奖牌。 该活动与“夏日支持你的公园”活动周末一起举行，一般被称为 “支持你的公园”年度牌匾活动。

POTA 文档的这一部分专门介绍了关于这个事件、其规则和结果的信息。
